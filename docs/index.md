# TwoDEngine Documentation

Welcome to the **TwoDEngine** documentation! TwoDEngine is a powerful 2D game engine built with modern C++ and designed for creating high-performance 2D games.

## 🎮 What is TwoDEngine?

TwoDEngine is a comprehensive 2D game development framework that combines the power of SDL2 for graphics and input, with modern tools like ImGui for debugging, Lua/Sol2 for scripting, and Python/pybind11 for extensibility. It provides a solid foundation for building 2D games with a focus on performance, flexibility, and ease of use.

## ✨ Key Features

- **SDL2 Integration**: Hardware-accelerated 2D rendering with SDL2
- **ImGui Support**: Built-in debugging and development UI
- **Scripting Support**:
  - Lua scripting with Sol2 for game logic
  - Python integration with pybind11 for tools and extensions
- **Math Library**: GLM (OpenGL Mathematics) for vector and matrix operations
- **Asset Management**: Support for images, fonts, sounds, and tilemaps
- **Cross-Platform**: Works on Windows, Linux, and macOS
- **Modern C++**: Built with C++17 standards

## 📋 System Requirements

### Minimum Requirements
- **OS**: Windows 10/11, Linux (Ubuntu 20.04+), macOS 10.15+
- **Compiler**: GCC 7+, Clang 6+, MSVC 2019+, MinGW-w64
- **RAM**: 4 GB
- **Storage**: 500 MB for engine and dependencies

### Recommended Requirements
- **OS**: Windows 11, Linux (Ubuntu 22.04+), macOS 12+
- **Compiler**: GCC 11+, Clang 14+, MSVC 2022+
- **RAM**: 8 GB or more
- **Storage**: 1 GB for engine, dependencies, and development tools

## 🛠️ Technology Stack

TwoDEngine is built on top of industry-standard libraries:

| Library | Purpose | Version |
|---------|---------|---------|
| **SDL2** | Graphics, Input, Audio | 2.0+ |
| **SDL2_image** | Image Loading | 2.0+ |
| **SDL2_ttf** | Font Rendering | 2.0+ |
| **SDL2_mixer** | Audio Mixing | 2.0+ |
| **ImGui** | Debug UI | 1.89+ |
| **GLM** | Math Library | 0.9.9+ |
| **Lua** | Scripting | 5.4+ |
| **Sol2** | Lua Binding | 3.3+ |
| **Python** | Extensibility | 3.10+ |
| **pybind11** | Python Binding | 2.11+ |
